To clone and run this application, you'll need [Git](https://git-scm.com),. From your command line:

```bash

$ git clone https://github.com/AugustoSavi/GymFitnessWebDesign.git

# Caminhe ate a pasta Web
$ cd GymFitnessWebDesign

$ Abra o index.html 
```
